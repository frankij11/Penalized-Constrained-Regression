Recommendations: When to Use Each Method

Use PCReg (Penalized-Constrained Regression) when:

  • Sample size is small (n_lots 10) - constraints provide valuable regularization
  • Prior knowledge is available about coefficient signs/bounds
  • Feature correlation is high ( 0.5) - constraints help with collinearity
  • Interpretability matters - ensures economically sensible coefficients

Standard OLS may be sufficient when:

  • Sample size is large (n_lots 30)
  • Feature correlation is low (< 0.3)
  • No prior constraints are appropriate for the application

Key Findings:

  • PCReg_CV_Tight (well-specified constraints) achieves the best performance when bounds are known
  • PCReg_CV_Wrong shows robustness - misspecified constraints still competitive
  • PCReg_ConstrainOnly often matches or beats PCReg_CV - penalty may not always help
  • OLS_LearnOnly (single predictor) performs poorly - rate effect is important
